/api - overview of each endpoint

/api/version/{version}/weather/temp - returns temperature fetched from other api
/api/version/{version}/weather/wind_speed - returns simulated wind speed

/api/version/{version}/economy/total_price/demand/{demand} - returns the total price of powering a house with the given demand and 
															 the amount of energy that the users wind turbine is providing.
/api/version/{version}/economy/market_price - returns the current electricity market price (kr/kWh)

/api/version/{version}/turbine/power - returns the amount of energy that the users wind turbine is currently generating


###/api/version/{version}/turbine/rpm - returns how fast the turbine is spinning
